# E6-S3: Weekly Review Snapshot

Epic: [Epic 6](../epics/EPIC-6-dashboard-and-performance-metrics.md)  
Priority: Release 2  
Size: S

## User Story

As a user, I want a compact weekly review so I can reflect on progress without digging through every session.

## Depends On

- E6-S1

## Unblocks

- repeat engagement with the insights layer

## Implementation Notes

- generate a weekly summary with wins, warnings, and suggested adjustments
- let users access current and prior weekly snapshots
- base the review only on recorded data

## Acceptance Criteria

- the app can generate a weekly summary view
- the summary includes wins, warnings, and one or two suggested adjustments
- past weekly reviews remain accessible
- the summary references actual logged data from that week

## Test And Verification

- weekly summary aggregation tests
- screenshot verification for weekly review layout
