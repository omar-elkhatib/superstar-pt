# Epic 7: Voice Notes and AI Summaries

## Goal

Reduce friction for richer feedback by letting users speak naturally, then review structured summaries before saving.

## Depends On

- Epic 3
- Epic 4
- a stable data model from Epic 0

## Unblocks

- Epic 8

## Stories

- [E7-S1 Record a voice note](../stories/E7-S1-record-a-voice-note.md)
- [E7-S2 Review and edit transcript summary](../stories/E7-S2-review-and-edit-transcript-summary.md)
- [E7-S3 Extract structured insights from voice](../stories/E7-S3-extract-structured-insights-from-voice.md)

## Scope

- voice recording during logging and follow-up
- transcript plus editable summary review
- structured insight extraction tied to saved records

## Likely Repo Touchpoints

- new voice capture module
- activity log and follow-up screens
- summary review UI
- future insight extraction service layer

## Exit Criteria

- users can attach and review voice notes without losing control of the data
- generated summaries are editable before save
- extracted fields are visible and correctable
